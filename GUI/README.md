## Clone du projet

Pour cloner le projet, executez la commande suivante

```shell script
git clone https://github.com/Afpa-repo/CDA-19224-1.git
```

Ou si vous voulez la version ssh

```shell script
git clone git@github.com:Afpa-repo/CDA-19224-1.git
```
